## 문제 3: 윤년 판별

연도를 입력받아 윤년인지 판별하세요.

**윤년 조건:**
- 4로 나누어떨어지면서 100으로 나누어떨어지지 않거나
- 400으로 나누어떨어지는 해

**입력:**
```java
int year = 2024;
```

**예상 출력:**
```
2024년은 윤년입니다.
```

**힌트:**
```java
(year % 4 == 0 && year % 100 != 0) || (year % 400 == 0)
```

## 문제 7: BMI 계산 및 판정

키(cm)와 몸무게(kg)를 입력받아 BMI를 계산하고 판정하세요.

**BMI 계산:** 몸무게(kg) / (키(m) × 키(m))

**판정 기준:**
- 18.5 미만: 저체중
- 18.5 이상 23 미만: 정상
- 23 이상 25 미만: 과체중
- 25 이상: 비만

**입력:**
```java
double height = 175.0;  // cm
double weight = 70.0;   // kg
```

**예상 출력:**
```
BMI: 22.86
정상 체중입니다.
```

## 문제 10: 할인율 계산

구매 금액에 따라 할인율을 적용하세요.

**할인 기준:**
- 10만원 이상: 10% 할인
- 5만원 이상: 5% 할인
- 5만원 미만: 할인 없음

**추가:** 회원이면 추가 5% 할인

**입력:**
```java
int price = 120000;
boolean isMember = true;
```

**예상 출력:**
```
원가: 120000원
할인율: 15%
최종 금액: 102000원
```
