## 문제 1: 배열 최댓값/최솟값

배열에서 최댓값과 최솟값을 찾으세요.

**입력:**
```java
int[] numbers = {45, 23, 78, 12, 89, 34};
```

**예상 출력:**
```
최댓값: 89
최솟값: 12
```

## 문제 2: 배열 뒤집기

배열을 역순으로 뒤집으세요.

**입력:**
```java
int[] arr = {1, 2, 3, 4, 5};
```

**예상 출력:**
```
원본: [1, 2, 3, 4, 5]
뒤집기: [5, 4, 3, 2, 1]
```

**힌트:** 두 개의 인덱스를 사용하여 양 끝에서 중앙으로 이동하며 교환

## 문제 3: 특정 값 찾기

배열에서 특정 값이 있는 인덱스를 찾으세요. 없으면 -1을 반환하세요.

**입력:**
```java
int[] arr = {10, 20, 30, 40, 50};
int target = 30;
```

**예상 출력:**
```
30의 인덱스: 2
```

## 문제 4: 배열 요소 개수 세기

배열에서 각 숫자가 몇 번 나오는지 세세요.

**입력:**
```java
int[] arr = {1, 2, 2, 3, 3, 3, 4, 4, 4, 4};
```

**예상 출력:**
```
1: 1개
2: 2개
3: 3개
4: 4개
```

## 문제 5: 2차원 배열 합계

2차원 배열의 모든 요소의 합을 구하세요.

**입력:**
```java
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};
```

**예상 출력:**
```
합계: 45
```


## 문제 6: 배열에서 두 번째로 큰 수

배열에서 두 번째로 큰 수를 찾으세요.
